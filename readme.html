La Linea — Gioco Tributo (B/N)

Fan-game didattico, mobile-first, ispirato al linguaggio grafico di La Linea. La baseline scorre, compaiono scalini/buchi/gobbe; tu aiuti il personaggio a saltare.

Omaggio stilistico all’idea di Osvaldo Cavandoli. Progetto non ufficiale e senza scopo di lucro.

Demo locale

Apri index.html in un browser moderno.

Comandi

  • Salto: tap sull’area “Tocca qui” oppure tasto Space
  • Pausa/Riprendi: bottone “Pausa” o tasto P
  • Restart: bottone “Restart” o tasto R

Struttura

/ (cartella del progetto)
├─ index.html
├─ app.js
├─ manifest.json
├─ sw.js
├─ icon-192.png
└─ icon-512.png

Installazione come PWA

  1. Lascia manifest.json e sw.js nella root.
  2. Pubblica su un hosting statico HTTPS (GitHub Pages/Netlify/etc.).
  3. Dal telefono: “Aggiungi alla schermata Home”.

Icone

  • icon-192.png — 192×192 px
  • icon-512.png — 512×512 px

Note tecniche

  • Generatori di ostacoli: step, gap, bump.
  • Collisione vs. forma locale della linea; cadere in un gap chiude la partita.
  • Difficoltà crescente con la velocità.

Personalizzazioni rapide

  • In app.js, modifica speed (velocità iniziale) e la frequenza di spawn (t % 70).
  • Ritocca lo stile del personaggio nella funzione guy.draw().

Deploy

GitHub Pages: repo → carica i file in main → Settings → Pages → Deploy from branch.

Server: copia la cartella su hosting HTTPS. Nessun build step.
